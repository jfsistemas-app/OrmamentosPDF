Política de Privacidade
Dados armazenados
O OrçaFácil armazena seus dados (orçamentos, clientes e dados da empresa) exclusivamente no seu dispositivo, utilizando SQLite local. Nenhuma informação é enviada a servidores externos.
Sem coleta de dados pessoais
Não coletamos, acessamos nem compartilhamos dados pessoais dos usuários. Todas as informações inseridas no app ficam somente no seu celular.
Anúncios
Na versão gratuita, exibimos vídeos de anúncios (rewarded ads) via AdMob. O Google pode coletar dados de uso conforme sua própria política de privacidade.
Backup
O recurso de backup exporta seus dados para um arquivo JSON salvo no seu próprio dispositivo. Você tem controle total sobre esse arquivo.
